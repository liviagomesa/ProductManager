Sobre o Gerenciador de Produtos

Este é um projeto desenvolvido para aprimorar o gerenciamento de inventário e produtos de forma prática e eficiente.

Funcionalidades principais:

  • 📋 Cadastro e edição de produtos
  • 🔍 Busca e filtragem dinâmica
  • 💾 Integração com API para persistência de dados
  • 🎨 Design moderno usando Angular Material
